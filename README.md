# Academic Review Skill — ИИ-рецензия научной статьи по критериям ВАК/Scopus

Скилл для структурированного рецензирования научных статей с помощью ИИ. На вход — статья (PDF или DOCX), на выходе — рецензионное заключение по 10 критериям уровня ВАК/Scopus, оформленное как документ Word, готовый к отправке редактору журнала.

Инструмент создан для **самопроверки автором** рукописи перед подачей в журнал: он работает как первый структурированный и беспристрастный рецензент, помогающий устранить технические и структурные недочёты до того, как статья попадёт к эксперту-человеку. ИИ-рецензия **не заменяет** решение редколлегии и работу анонимного рецензента.

## Метод

В основе — авторский подход, описанный в статье **«Как использовать нейросеть для рецензии научной статьи»**: [inna-gazieva.ru/blog/ai-recenzent-nauchnoj-stati](https://inna-gazieva.ru/blog/ai-recenzent-nauchnoj-stati/)

Ключевая идея: простой запрос «отрецензируй мою статью» даёт обобщённый и малополезный результат. Чтобы получить содержательную рецензию, модели задаётся структура работы приглашённого рецензента академического журнала — ролевая установка, явный чек-лист критериев и формат итогового заключения.

## Десять критериев экспертизы

1. Название статьи
2. Актуальность
3. Научная новизна
4. Теоретическая база и обзор литературы
5. Методология
6. Достоверность и обоснованность результатов
7. Структура и логика изложения
8. Аннотация и ключевые слова
9. Оформление (ГОСТ Р 7.0.100-2018 / APA)
10. Этические аспекты

По каждому критерию скилл даёт оценку с опорой на конкретные фрагменты текста, формирует перечень обязательных и рекомендуемых правок и выносит итоговое решение из четырёх стандартных вариантов — от «рекомендовать к публикации» до «отклонить».

## Как использовать

Скилл рассчитан на среды, поддерживающие формат Agent Skills (например, Claude). Подробные шаги описаны в [`SKILL.md`](./SKILL.md).

Кратко:
1. Загрузите научную статью (PDF или DOCX).
2. Попросите провести рецензирование.
3. Получите рецензионное заключение в формате DOCX.
